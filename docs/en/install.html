Install Guide

Prerequisites

Node.js > 8 required.

Install to global

You can install jqf command with npm or yarn.

with npm

$ npm install jqf --global

with yarn

$ yarn global add jqf

Instant execution with npx

$ npx jqf -h

Quick Start with npx

You can try jqf instantly with npx.

$ echo '{"hello": "world"}' | npx jqf 'obj => obj.hello'
"world"
